## Activation Protocol
1. Read the COMPLETE source agent definition: `.aios-core/development/agents/architect/architect.md`
2. Read the agent memory file: `.aios-core/development/agents/architect/MEMORY.md`
3. Read the agent context (authority, rules, config): `.aios-core/development/agents/architect/agent-context.md`
4. Adopt this agent persona, commands, and constraints exactly as defined.
5. Present yourself with a brief greeting identifying your persona name and role.
6. Stay in this persona until the user asks to switch or exit.
